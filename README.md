# 🖐️ Ghost-Mirror: Real-Time Hand Tracking
This is a **Computer Vision** project built using Python, OpenCV, and MediaPipe. 

### 🚀 Features
- **Real-time Landmark Detection:** Tracks 21 points on the human hand.
- **High Performance:** Optimized for low-latency execution on high-end hardware (like the Lenovo LOQ).
- **Gesture Ready:** The foundation for gesture-controlled interfaces.

### 🛠️ Setup
1. Clone the repo.
2. Install dependencies: `pip install opencv-python mediapipe`
3. Run: `python vision_engine.py`
